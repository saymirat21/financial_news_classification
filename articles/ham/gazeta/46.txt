Минфин: доход по ОФЗ для россиян не будут облагать подоходным налогом 
Доход по облигациям федерального займа для населения не будет облагаться подоходным налогом. Об этом сообщило ТАСС со ссылкой на пресс-службу Минфина.
«Купонный доход по ОФЗ для населения не будет облагаться НДФЛ», — заявили в ведомстве.
Ранее в Госдуме назвали неактуальной идею Минфина об облигациях для населения.
В свою очередь глава Минфина Антон Силуанов сообщил о готовности инвестировать свои средства в облигации для населения.
